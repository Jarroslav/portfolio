# Portfolio

Single-page portfolio / CV site for Yaroslav Krivushenko — Test Automation Architect (Agentic SDLC / Quality Engineering).

Built with Vite + React + TypeScript. Deployed to GitHub Pages via GitHub Actions on every push to `main`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Assets

- `public/images/portrait.svg` — placeholder portrait; replace with a real photo at `public/images/portrait.jpg` (and update the `src` in `src/components/Hero.tsx`) when available.
- `public/images/from-ashes.png` — real screenshot of [from-ashes.com](https://www.from-ashes.com).
